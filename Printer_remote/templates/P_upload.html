PDFs are best supported but other files that work are

  • Text Documents: txt, docx, doc (Microsoft Word Documents), and other plain text or formatted documents
  • Presentations: ppt, pptx (Microsoft PowerPoint Presentations)
  • Some images like .jpg .png .jpeg .gif
  • Spreadsheets: xls, xlsx (Microsoft Excel Spreadsheets)

Consider converting the file you wish to print to PDF before submitting if it fails
